## 📌 專案介紹

本專案為一個基於 **Spring Boot** 的入門練習專案，
透過簡單的 REST API，實現基本的後端服務運作流程。

主要目的是熟悉：

* Spring Boot 專案架構
* RESTful API 建立方式
* Maven 專案管理

---

## 🚀 功能說明

* 提供一個簡單的 API：

  * `GET /` → 回傳文字訊息
* 測試 Spring Boot 啟動與運行流程

---

## 🛠️ 使用技術

* Java 21
* Spring Boot 3.5.4
* Maven

---

## 📂 專案結構

```
src/main/java/
└── training/
    └── web/
        └── Hello.java
```

---

## 📖 學習重點

* Spring Boot 基本設定
* `@SpringBootApplication` 使用
* `@RestController` 建立 API
* `@GetMapping` 處理 HTTP GET 請求
